# Classic ML — конспект

## 1. Что такое Classic ML

**Classic Machine Learning** — это традиционные методы машинного обучения, которые не используют большие нейросети.
Цель: извлечь закономерности из данных и делать предсказания.

Примеры задач: классификация, регрессия, кластеризация, снижение размерности.

---

## 2. Основные методы

### 2.1 Линейная регрессия

* Предсказывает числовое значение
* Формула:

```
y = w1*x1 + w2*x2 + ... + b
```

* Loss: MSE (mean squared error)

### 2.2 Логистическая регрессия

* Классификация (0/1)
* Sigmoid функция:

```
P(y=1) = 1 / (1 + exp(-z))
```

* Loss: Binary Cross-Entropy

### 2.3 Деревья решений

* Разбивает данные по признакам
* Плюсы: интерпретируемость
* Минусы: переобучение без ограничения глубины

### 2.4 Random Forest / Gradient Boosting

* Ансамбли деревьев
* Random Forest → bagging
* Gradient Boosting → последовательное исправление ошибок

### 2.5 KNN (k-nearest neighbors)

* Предсказывает на основе ближайших точек
* Простая идея, плохо масштабируется при больших данных

### 2.6 SVM (Support Vector Machine)

* Ищет гиперплоскость, разделяющую классы
* Можно использовать kernel trick для нелинейных границ

### 2.7 K-Means

* Кластеризация данных
* Разбивает объекты на k кластеров, минимизируя внутрикластерное расстояние

---

## 3. Feature Engineering

* Выбор признаков, нормализация, преобразование категориальных данных
* Примеры:

  * StandardScaler / MinMaxScaler
  * OneHotEncoding / LabelEncoding
  * Polynomial features

---

## 4. Метрики оценки

### Для классификации:

* Accuracy
* Precision / Recall / F1-score
* Confusion matrix

### Для регрессии:

* MSE, RMSE
* MAE
* R² score

### Для кластеризации:

* Silhouette score
* Davies-Bouldin index

---

## 5. Обучение и валидация

* Train / Validation / Test split
* Cross-validation (K-fold)
* Grid search / Random search для подбора гиперпараметров

---

## 6. Плюсы Classic ML

* Легко интерпретировать
* Работает на небольших данных
* Быстро обучается

## 7. Минусы Classic ML

* Плохо масштабируется на большие данные и сложные задачи
* Ограниченные возможности для обработки текста и изображений без feature engineering
* Не умеет самообучаться на новых данных без дообучения

---

## 8. Интуитивно

Classic ML = «классические инструменты статистики с автоматическим обучением»:

* На вход → признаки (features)
* Модель → закономерности
* Выход → предсказания

---

# Краткое резюме

Classic ML — основа машинного обучения:

* Линейная / логистическая регрессия
* Деревья и ансамбли
* KNN, SVM, K-Means
* Метрики: accuracy, recall, MSE, silhouette

Подходит для небольших и средних задач, где данные структурированы и не требуется глубокое обучение.
